匈牙利算法是一种组合优化算法，用于求解指派问题(assignment problem)，算法时间复杂度为O(n3)O(n3)。Harold Kuhn发表于1955年，由于该算法基于两位匈牙利数学家的早期研究成果，所以被称作“匈牙利算法”。
